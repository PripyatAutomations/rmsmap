MAP HELP
--------

[Icons]
  +         Zoom In
  -         Zoom Out
  |->       Measure a route
  X         Stop/clear route
  km/mi/nm  Set unit

[Mouse]
  * Right click - Hide/show magnifier
     - L.Click: Toggle markers (NYI)
  * Zoom: Double click point of interest or use scrollwheel

[Touch]
  * Normal touch squeeze/stretch/pan controls

[Keyboard]
  * Pan/Zoom *
  Arrow Keys	Pan
  + / -	   	Zoom

  * Measuring *
  ESC		End route
